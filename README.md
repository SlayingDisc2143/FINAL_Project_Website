# Tyler Wittie – Computer Maintenance Website

## Project Overview

This project is a professional computer maintenance website built using HTML and CSS. The homepage serves as both an introduction and an About Me section, while additional pages expand on the services I offer and the tools I use in my work.

The site was developed as a school project, but it is structured as a real-world professional portfolio. It demonstrates coding structure, consistent styling, navigation design, and basic front-end development skills.

## Purpose

The purpose of this website is to:

- Showcase my interest and growth in IT and computer maintenance  
- Present my service offerings clearly and professionally  
- Demonstrate structured HTML and CSS development  
- Apply project management principles to a technical build  

Rather than treating this as just an assignment, I approached it as a functional portfolio piece.

## Site Structure

### Root Directory

- `index.html` – Homepage (Introduction + About Me)
- `services.html` – Service tiers and computer build options
- `tools.html` – Tools and equipment I use
- `style.css` – Shared external stylesheet
- `scope.md` – Final revised project scope
- `plan.md` – Project plan including TAME risk analysis and task schedule
- `retrospective.md` – Project reflection and lessons learned

## Technical Features

- Single shared external CSS file for consistent styling  
- Responsive layout adjustments for smaller screens  
- Card-based layout for services  
- Flexbox layout for tools section  
- Image references included for transparency  
- Clean navigation across all pages  

## Project Management Integration

This project incorporates principles from *Project Management for the Unofficial Project Manager*, including:

- Defined scope  
- Risk planning using TAME  
- Structured task breakdown  
- Final retrospective evaluation
